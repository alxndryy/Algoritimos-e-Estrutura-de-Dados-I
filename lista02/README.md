## questão 5:

## questão 6:

C não seja divisível nem por A nem por B, imprima na tela “Não é divisível”. Caso o usuário digite
um valor negativo ou zero para A e/ou B, imprima na tela “Valor inválido” e finalize a execução
do programa."*
